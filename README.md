# Nashville Notation Wheel

An interactive music theory tool that visualizes key relationships, chord progressions, guitar chord diagrams, and piano keyboard layouts simultaneously.

Built in vanilla HTML5. No dependencies. Works in any browser.

## What It Does

- Select any key on the chromatic wheel
- - Choose chord degrees using Roman numerals (I through VII)
  - - See guitar chord diagrams and piano keyboard update in real time
    - - Color coding shows each note's relationship to the root
     
      - ## Why
     
      - Nashville Number System is how professional musicians communicate chord progressions quickly across any key. This tool makes that system visual and immediate for learners at any level.
